# Customer Churn Prediction using PySpark
📌 Project Overview

This project focuses on predicting customer churn using PySpark and Machine Learning.

The goal is to identify customers who are likely to leave a telecom service, helping businesses take proactive actions.

🎯 Problem Statement

Customer churn leads to major revenue loss.

This project builds a binary classification model to predict:

Churn → Yes / No
⚙️ Tech Stack

Apache Spark MLlib

Pandas

Matplotlib / Seaborn

Telco Customer Churn Dataset

Contains customer demographics, services, and billing info
